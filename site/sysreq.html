Audio Unit plugins for macOS

Our Audio Unit plugins require macOS 10.9 ("Mavericks") or higher. They run natively on Apple silicon Macs and 64-bit Intel-based Macs. The host must support AUs using the Audio Component API (introduced in 2009).

VST plugins for 64-bit Windows

Our 64-bit VST plugins for Windows should work with any x86-64 processor on a 64-bit version of Windows, but we are only testing on Windows 10. There are some Windows 10 APIs that we would like to use, so we may add this as a requirement in the future.

VST plugins for old 32-bit Windows

The classic 32-bit VST plugins still work (including on 64-bit machines) and might be a good choice for older systems, but we are no longer updating those.

VST plugins for Mac OS X

Our VST plugins for Mac OS X run on Mac OS X versions 10.1 ("Puma") through 10.6 ("Snow Leopard"). They are CFM code format for PowerPC processors. They may run under the "Rosetta" environment for Intel processors, but they do not contain native x86 executable code, nor are they available in the native Mac OS X code format Mach-O.

VST plugins for old Mac OS

Our software for classic Mac OS runs on Mac OS versions 8.5 through 9.2.2.
