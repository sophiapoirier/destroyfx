EQ Sync

rate:

The rate parameter lets you control EQ Sync's rate according to the tempo of your song. The values are in cycles per beat (note: I mean per single beat, not per measure, so this is time signature independent).


smoothing:

The smoothing parameter controls what percentage of each EQ cycle is spent gradually morphing into the next EQ setting.


tempo:

The tempo parameter is used to tell EQ Sync what the tempo of your song is. You can alternatively enable the "host sync" option, which means that EQ Sync tries to get the tempo and beat position from the host application.

Please note that "host sync" for tempo only works if the host application supports sending tempo information to plugins. If "host sync" is enabled and the host supports it, then the tempo parameter's value is ignored (since the host's tempo is being used instead).
